    The timeout to use for database server operations, in seconds
    System.Int32 batchSize

    The batch size to use when copying rows to the destination. If 0, all rows are streamed in a single operation. If non-zero, rows are sent in batches, each in their own transaction.
    Remarks

    The default batchSize value of 0 provides the fastest copy, but a non-zero value may be needed if the size or duration of server operations must be kept below a minimum.
